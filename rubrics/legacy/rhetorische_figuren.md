# Rhetorische Figuren – Referenztabelle

Wird verwendet zur Beurteilung von Ausdruck und Stilmitteln in allen Textsorten.

## Figuren

| Figur               | Beschreibung                                                                                      | Beispiel                                                                 |
|---------------------|---------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| Akkumulation        | Anhäufung von ähnlichen Begriffen oder Wörtern                                                    | Sonne, Mond und Sterne, Rosen, Tulpen, Nelken                           |
| Alliteration        | Mindestens zwei aufeinanderfolgenden Wörter haben denselben Anfangslaut                           | Kind und Kegel, offen sagt, Geiz ist geil                               |
| Ellipse             | Auslassung eines Satzteils bzw. eines unvollständiger Satzes                                      | Was nun? Erst die Arbeit, dann das Vergnügen.                           |
| Hyperbel            | Starke Übertreibung                                                                               | Schneesturmtempo, Todmüde, stocktaub, himmelhochjauchzend, zu Tode erkältet |
| Interjektion        | Kurzer Ausruf, Gefühlsausdruck                                                                    | Oh!, Aha!, Hey!, Phat!, Okay!, Ey!                                      |
| Ironie              | Aussage, die eigentlich das Gegenteil von dem meint, was tatsächlich gesagt wird; oft sarkastisch | Schöne Bescherung! Toll gemacht!                                        |
| Klimax              | Dreigliedrige Steigerung                                                                          | Heute back ich, morgen brau ich, übermorgen hol ich mir die Königskind  |
| Metapher            | Sprachlicher Ausdruck, bei dem ein Wort (oder eine Wortgruppe) aus seinem eigentlichen Bedeutungsrahmen herausgenommen und in einen anderen Bedeutungszusammenhang übertragen wird; bildliche Übertragung | rosarote Brille aufhaben, ein Meer von Menschen, eine Wärmebombe, eine Glückssträhne haben |
| Neologismus         | Wortneuschöpfung, sprachliche Neubildung von Wörtern                                             | Hopster, Froilginn                                                       |
| Oxymoron            | Zwei sich widersprechende Begriffe                                                                | offenes Geheimnis, bitterstöß, Hassliebe, heißkalt                     |
| Paradoxon           | Aussage mit scheinbarer Widersprüchlichkeit                                                       | Ich weiß, dass ich nichts weiß. Das Leben ist der Tod, und der Tod ist das Leben. |
| Parallelismus       | Wiederholung derselben Wortreihungsfolge                                                          | Keine Kinder, keine Sorgen – große Kinder, große Sorgen. Herb ist das Leben, kalt ist der Schnee. |
| Parenthese          | Einschub von ergänzenden, informativen Wörtern oder Wortgruppen; Kennzeichnung: – oder ()        | Ich bin – wie gesagt – schwer verliebt.                                 |
| Personifikation     | Vermenschlichung von Dingen, Begriffen, Tieren, Nichtmenschlichem; Art der Metapher              | Mutter Natur, Der Finanzmarkt ist eingebrochen, Die Sonne lacht.        |
| Rhetorische Frage   | Scheinfrage, die eine bereits bekannte Antwort suggeriert                                         | Für wie blöd hältst du mich eigentlich? Ist das dein Ernst?             |
| Symbol              | Sinnbild, das etwas Abstraktes, oft Unanschauliches verdeutlicht                                 | rotes Herz für Liebe, weiße Taube für Frieden, Koffer für Reise, Glühbirne für Idee |
