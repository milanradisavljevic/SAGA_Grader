# SETUP-PROMPT – Ordnerstruktur erstellen

Führe folgende Aufgaben aus:

1. Erstelle im aktuellen Verzeichnis folgende Ordnerstruktur:

```
./input/
./output/
./rubrics/
```

2. Erstelle in ./output/ eine leere Datei namens fehlerlog.txt

3. Kopiere folgende Dateien aus dem Verzeichnis, in dem du dich befindest, in den Ordner ./rubrics/:
   - eroerterung.md
   - kommentar.md
   - aufbau_referenz.md
   - rhetorische_figuren.md

4. Kopiere die Datei MASTER_PROMPT.md in das Hauptverzeichnis (bleibt dort, wo sie ist).

5. Bestätige am Ende:
   - Welche Ordner wurden erstellt
   - Welche Dateien liegen in ./rubrics/
   - Ob ./output/fehlerlog.txt existiert
   - Ob ./input/ leer ist und bereit für Schülerarbeiten

Führe keine weiteren Aktionen durch. Warte nach der Bestätigung auf neue Anweisungen.
